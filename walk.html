Manifesto


I wanted to start making 3D models and simple video games. But to be honest when I looked into tutorials a lot of it looked like a black box, which can be ok for some folks but I prefer to understand in detail what I am doing. Therefore I made a decision to start from the maths and implement a simple 3D graphics engine first. It is written in Typescript and uses HTML 2d canvas as base for rendering. I am planning to add support for OBJ files, cameras, light sources, etc.
